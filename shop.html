■ Product Description

- [Traveler Edition], [Explorer Edition], and [Conqueror Edition] are packages that grant access to Black Desert, and contain items that can be used in-game.

- Each package contains items that will be immediately sent to your account once the package is purchased.

- Each package contains items that will be sent to the account that made the purchase and cannot be deleted.

- Each item’s description can be viewed in the shop and/or detailed item description in-game.

- If you purchase a package on an account that already has access to Black Desert, you will be sent a separate web coupon code for Black Desert game access which can be found on your My Page.

- Each account can only purchase a package product ([Traveler Edition], [Explorer Edition], [Conqueror Edition]) once, and multiple/duplicate packages cannot be purchased.

- Packages cannot be gifted or transferred.

- Packages may be changed after prior notice.

- The items included in each package will be sent to your Web Storage after receiving the package on My Page.

- The items included in each package may have restrictions on trading/selling/usage periods and different binding settings. Due to these reasons, the items cannot be transferred or restored.

- If an account or character is restricted from the game due to violating the Operational Policy, there will be no refunds for any purchased packages due to restrictions to access the game.

- Inventory Slot Expansion Coupons cannot be used on a character that has expanded all of the available inventory slots (192 slots).

- Unless otherwise stipulated by the relevant law/regulation and/or explicitly in the service clause, purchased packages and the items included in these packages are not eligible for a refund once finalized.
